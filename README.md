# Startup-Founder-Dataset
Startup Founder Dataset Analysis Dashboard
Overview

This project presents an interactive Power BI dashboard built using a Startup Founder Dataset to analyze the characteristics, backgrounds, and success patterns of startup founders.

The dashboard provides insights into:

Founder experience levels
Startup success rates
Educational background impact
Big Tech experience influence
Previous startup experience
Sales and product management background
University influence on startup outcomes

The project is designed for:

Data analytics practice
Startup ecosystem research
Business intelligence portfolio projects
Founder success trend analysis

Dataset Information

The dataset contains startup founder information including:

Column	Description
Experience_Level	Founder seniority level
Success	Startup success indicator
Previous Startups	Number of prior startups
Elite_Edu	Elite education flag
Tech_Background	Technical background indicator
Worked at Google	Google employment history
Worked at Microsoft	Microsoft employment history
Worked in Sales	Sales experience
Product Leadership	Product management/director/VP background
University	Founder university
Key Insights

Some example findings from the dashboard:

Senior founders show higher startup success rates.
Big Tech experience positively correlates with founder success.
Founders from elite universities tend to have slightly higher success metrics.
Previous startup experience improves probability of success.
Product leadership backgrounds are common among successful founders.

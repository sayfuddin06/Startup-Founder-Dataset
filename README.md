# Startup Founder Dataset Analysis Dashboard

## Executive Summary

This project delivers a professional **Business Intelligence dashboard** built in **Power BI** to analyze startup founder profiles, educational backgrounds, professional experience, and their relationship to startup success.

Using a structured startup founder dataset, the dashboard uncovers patterns that influence entrepreneurial outcomes and provides data-driven insights into founder characteristics associated with high-growth startups.

The solution combines:

* Interactive data visualization
* Business intelligence reporting
* Founder segmentation analysis
* Startup success metrics
* Executive-level KPI monitoring

The dashboard was designed to simulate a real-world analytics solution suitable for:

* Venture capital research
* Startup ecosystem analysis
* Founder benchmarking
* Business intelligence portfolios
* Data analytics case studies

---

## Dashboard Objectives

The primary objectives of this project are:

* Analyze how founder experience impacts startup success
* Evaluate the influence of elite education and university background
* Measure the effect of Big Tech employment on startup outcomes
* Identify patterns among high-growth founders
* Deliver executive-ready startup analytics through Power BI

---

## Key Business Questions

This dashboard helps answer the following analytical questions:

* Do experienced founders achieve higher startup success rates?
* Does working in Big Tech improve entrepreneurial outcomes?
* Are founders from elite universities more successful?
* How does previous startup experience influence growth?
* What founder backgrounds are most common among successful startups?

# Dataset Information

The dataset contains startup founder information including:

| Column              | Description                               |
| ------------------- | ----------------------------------------- |
| Experience_Level    | Founder seniority level                   |
| Success             | Startup success indicator                 |
| Previous Startups   | Number of prior startups                  |
| Elite_Edu           | Elite education flag                      |
| Tech_Background     | Technical background indicator            |
| Worked at Google    | Google employment history                 |
| Worked at Microsoft | Microsoft employment history              |
| Worked in Sales     | Sales experience                          |
| Product Leadership  | Product management/director/VP background |
| University          | Founder university                        |

---

# Key Insights

Some example findings from the dashboard:

* Senior founders show higher startup success rates.
* Big Tech experience positively correlates with founder success.
* Founders from elite universities tend to have slightly higher success metrics.
* Previous startup experience improves probability of success.
* Product leadership backgrounds are common among successful founders.
